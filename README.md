# Student Grade Tracker

## About the Project

Student Grade Tracker is a Java-based console application developed as part of my CodeAlpha Java Programming Internship.

The project helps manage student records and grades efficiently. Users can add students, record marks, view student information, generate reports, and track overall class performance. The application also uses file handling to save data, ensuring that records are not lost when the program is closed.

---

## Features

* Add new students
* Add grades for students
* View individual student details
* View all student records
* Calculate average marks
* Find highest and lowest scores
* Generate class performance reports
* Display student rankings
* Delete student records
* Save and load data using file handling

## How to Run

1. Clone or download the repository.
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Make sure Java is installed on your system.
4. Run the Main.java file.
5. Use the menu options to manage student records and grades.
